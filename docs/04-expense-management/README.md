This section covers all expense-related functionality including creation, splitting options, categories, and operations.

## Feature Overview

### Expense Creation
- Add expense with amount and description
- Select who paid (payer)
- Select who participated (owes money)
- Attach receipt or image
- Assign category

### Split Options
- **Equal Split**: Divide evenly among participants
- **Unequal Split**: Specify exact amounts per person
- **Split by Shares**: Proportional split based on shares
- **Split by Percentage**: Percentage-based split

### Categories
- Predefined categories per group type
- Custom category support
- Category statistics

### Operations
- View expense list with filters
- View individual expense details
- Edit expense (with permissions)
- Delete expense (with permissions)
- Receipt/image management
