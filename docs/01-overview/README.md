## Quick Summary

The Expense Splitting App is a web application that enables groups of users to:
- Track shared expenses
- Split costs using various methods
- Calculate who owes whom
- Record and approve offline payments
- Maintain transparency through audit trails

The app follows a modern full-stack architecture with FastAPI backend, React frontend, and PostgreSQL database.
