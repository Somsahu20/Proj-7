# 02. User Management

This section covers all user-related functionality including authentication, profile management, and notification preferences.

## Section Contents

| Document | Description |
|----------|-------------|
| [Authentication](./authentication.md) | Email/password signup, Google OAuth, password reset, JWT handling |
| [Profile Management](./profile-management.md) | User profile CRUD, profile picture, settings |
| [Notification Preferences](./notification-preferences.md) | Email notification settings and digest options |

## Feature Overview

### Authentication
- Email/password registration and login
- Google OAuth 2.0 social login
- Password reset via email
- JWT-based session management
- Secure token refresh mechanism

### Profile Management
- View and edit personal information
- Upload profile picture
- Update email and phone
- Manage account settings

### Notification Preferences
- Toggle individual notification types
- Configure email digest frequency
- Set reminder preferences
