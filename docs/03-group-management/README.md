# 03. Group Management

This section covers all group-related functionality including creation, member management, and settings.

## Section Contents

| Document | Description |
|----------|-------------|
| [Group Creation](./group-creation.md) | Creating groups with categories and initial settings |
| [Member Management](./member-management.md) | Inviting, adding, removing members and role management |
| [Group Settings](./group-settings.md) | Editing, deleting groups and viewing statistics |

## Feature Overview

### Group Creation
- Create group with name, description, and image
- Select group category (trip, apartment, couple, etc.)
- Set initial members
- Creator becomes admin automatically

### Member Management
- Invite members via email or username
- Add/remove members (admin only)
- Assign roles (Admin/Member)
- Transfer admin role

### Group Settings
- Edit group details (name, description, image, category)
- Delete group (with settlement requirements)
- View group statistics and activity
